# Customer-Behavior-Analysis
The objective of this project is to analyze customer purchasing behavior and generate meaningful business insights using structured data. The project focuses on understanding customer demographics, revenue patterns, product performance, and  segmentation.

# Dataset Overview
The dataset used in this project contains customer transaction data. It includes 
information related to customer demographics, purchasing behavior, and product 
details.
Key columns in the dataset include:
Customer ID – Unique identifier for each customer 
Age – Age of the customer 
Gender – Male/Female classification 
Item Purchased – Product bought by the customer 
Category – Product category 
Purchase Amount USD – Amount spent on purchase 
Location – Customer’s geographical location 
Subscription Status – Whether the customer is subscribed 
Shipping Type – Mode of delivery 
Review Rating – Customer rating for product 
Previous Purchases – Number of past purchases 
The dataset was checked for inconsistencies and prepared for analysis.

# Tools & Technologies Used
SQL (MySQL): Used for data cleaning, aggregation, and analysis 
Power BI: Used for data visualization and dashboard creation 
Excel/CSV: Used as the data source

# Data Analysis (SQL)
Data cleaning was performed using SQL to ensure the dataset was consistent and reliable for analysis. SQL was used to perform various analytical operations to extract insights from the dataset.
The following analyses were conducted:
Calculated the average age of customers 
Analyzed gender distribution to understand customer base 
Computed total revenue generated by each gender
Evaluated the impact of subscription status on revenue
Identified most frequently purchased products
Calculated average product ratings
Segmented customers into: 
o New customers 
o Returning customers 
o Loyal customers 
Analyzed spending behavior across different shipping types
Grouped customers into age categories (Young, Adult, Senior) and 
calculated revenue

# Key Insights
The following insights were derived from the analysis:
Male customers contribute a higher portion of total revenue compared to 
female customers 
Non-subscribed customers generate more revenue than subscribed customers 
Loyal customers form the largest segment of the customer base 
A small number of products dominate overall purchases 
Spending patterns are relatively consistent across different shipping types 
Adult customers contribute the highest revenue among all age groups 
These insights can help businesses improve targeting and decision-making.

# Conclusion
This project demonstrates how SQL and Power BI can be combined to analyze 
structured data and build an interactive dashboard. The insights derived from the 
data can support business decisions related to customer targeting, product strategy, 
and revenue optimization.
